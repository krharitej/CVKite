# CVKite – Resume Builder Platform

CVKite is a full-stack resume builder platform that enables users to create, customize, and preview professional resumes in real time. The application provides multiple resume templates, section-based editing, and export-ready formats to help users generate ATS-friendly resumes efficiently.

The platform also integrates secure online payments using Razorpay, allowing users to access premium templates and advanced customization features. CVKite is designed with a focus on usability, scalability, and performance, providing a smooth end-to-end experience from resume creation to final download.


## Features

- Real-time resume preview
- Multiple customizable templates
- Section-wise resume editing
- Secure Razorpay payment integration
- Premium feature access
- Export-ready resume formats
- ATS-friendly layout support


## Tech Stack

- Frontend: React.js
- Backend: Spring Boot
- Build Tool: Maven
- Database: MongoDB
- Payment Gateway: Razorpay
- Authentication: JWT / Session Management
- Styling: CSS / Tailwind CSS
- CI/CD: GitHub Actions


## System Architecture

```
   User Interface (React)
            ↓
 REST APIs (Spring Boot)
            ↓
     Database (MongoDB)
            ↓
 Payment Gateway (Razorpay)
```
